# SwitchStyle

A Steem App to schedule users styles switches, by setting the frequency and unity ( minutes , hours or days)

beta

------------------------------------------------------------------------------

To Install 
- git clone this repo
- cd switchstyle\server 
- npm install
- node index.js (start server)
- cd switchstyle\client
- npm install
- npm start (start client)

* the Server needs an instace of mongodb running at the default port 27017
